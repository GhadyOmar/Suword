## 🧠 Overview

**Suword** is a logic puzzle game that enhances traditional Sudoku gameplay by introducing interconnected grid mechanics.

Unlike classic Sudoku, Crossdoku requires players to solve overlapping sections that influence each other — increasing complexity and encouraging deeper logical thinking.

### 🎯 Objective

- Fill all empty cells correctly
- Avoid duplicate numbers in rows and columns
- Solve interconnected grid sections logically
- Complete the puzzle with accuracy and focus

## 🏗 Architecture

The app follows the **MVVM (Model–View–ViewModel)** architectural pattern.

### Why MVVM?

- Clean separation of concerns
- Scalable and maintainable structure
- Clear state management
- Testable logic components
